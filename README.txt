== Description ==

Popup Manager is the most powerful free popup builder available. With it you can easily create attractive popups and manage their style, dimensions, effects, triggers and much more with no coding skills required.

Popup Manager is a great tool for your marketing campaign that allows you to build unlimited amount of highly customizable popup windows and add them to any page you like. Each popup comes with advanced styling options where you can specify things like font, margin, background, close icon, colors, and much more.

== Installation ==

Installing "Popup manager" can be done either by searching for "Popup manager" via the "Plugins > Add New" screen in your WordPress dashboard or by using the following steps:

1. Download the plugin via WordPress.org
1. Upload the ZIP file through the 'Plugins > Add New > Upload' screen in your WordPress dashboard
1. Activate the plugin through the 'Plugins' menu in WordPress
